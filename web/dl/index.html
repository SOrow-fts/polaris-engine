    Linux/Chromebook

    お手軽:

    wget https://raw.githubusercontent.com/ktabata/polaris-engine/master/install-polaris-engine.sh && sh ./install-polaris-engine.sh
    poaris-engine

    本格的:

    git clone https://github.com/ktabata/polaris-engine.git
    cd polaris-engine
    make
    sudo make install
    polaris-engine
